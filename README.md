# 🐾 Tamagochi Clicker

Учебный консольный ООП-проект на Python.

Проект представляет собой модифицированную версию классического «Тамагочи»: это не только симулятор питомца, но и небольшой кликер. Игрок следит за состоянием питомца, кормит его, лечит, даёт отдыхать и зарабатывает монеты для покупки еды и лекарств.

## 🎯 Цель проекта

Проект был выполнен в рамках учебного задания по ООП.

Основная цель — реализовать игровую логику через классы, интерфейсы и разделение ответственности между компонентами.

В проекте отрабатывались:

* объектно-ориентированное программирование;
* абстрактные базовые классы;
* наследование;
* композиция объектов;
* декомпозиция проекта на модули;
* кастомные исключения;
* игровой цикл.

## 💡 Особенности задания

По исходному заданию проект должен был быть не просто обычным Тамагочи, а гибридом:

* **Тамагочи** — питомец имеет состояние: голод, здоровье, энергию.
* **Кликер** — игрок зарабатывает монеты, чтобы покупать еду и лекарства.
* **Игровая сущность** — управляет ресурсами, инвентарём и взаимодействием между компонентами.

Интересный нюанс задания: основная логика игры должна быть отделена от конкретного способа запуска. То есть классы игры не должны зависеть от того, работает игра в консоли, окне или другом интерфейсе.

## 🎮 Возможности игры

* создание виртуального питомца;
* заработок монет;
* покупка еды;
* покупка лекарств;
* кормление питомца;
* лечение питомца;
* отдых питомца;
* игры с питомцем;
* изменение состояния питомца после действий;
* завершение игры при гибели питомца.

## 🧩 Основные компоненты

Проект разделён на несколько сущностей:

* `Tamagochi` — логика питомца и его состояния;
* `Food` — модель еды;
* `Medicine` — модель лекарства;
* `Clicker` — логика заработка монет;
* `Game` — управление игровым процессом, инвентарём и ресурсами;
* `exceptions` — кастомные исключения для ошибок игры.

## ⚙️ Установка

### Клонирование репозитория

```bash
git clone https://github.com/VirentLilium/higher-front-back-dev-tamagochi.git
cd higher-front-back-dev-tamagochi
```

### Создание виртуального окружения

```bash
python -m venv venv
```

### Активация виртуального окружения

Windows:

```bash
venv\Scripts\activate
```

Linux / macOS:

```bash
source venv/bin/activate
```

### Обновление pip и установка зависимостей

```bash
python -m pip install --upgrade pip
pip install -r requirements.txt
```

## ▶️ Запуск проекта

```bash
python main.py
```
